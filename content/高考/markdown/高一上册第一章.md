# 微积分笔记

## 导数定义

导数的定义公式：

$$f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h}$$

### 常见导数公式

1. 基本导数：
   - $(x^n)' = nx^{n-1}$
   - $(e^x)' = e^x$
   - $(\ln x)' = \frac{1}{x}$

## 代码示例

```python
import numpy as np

def derivative(f, x, h=1e-7):
    return (f(x + h) - f(x)) / h
```
